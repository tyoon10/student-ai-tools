- **Previous validation pass:** 2026-05-03 (initial)

## Per-entry schema

For each validated tool:

1. **Offered tier and pricing** — original price, student price (free or discounted).
2. **Included features** — what the student tier actually unlocks.
3. **Effective length** — duration, renewal behavior, re-verification cadence.
4. **Claim path type** — Individual-claimable: Yes / No (if No, capture institution check path only).
5. **Verification method** — SheerID, .edu email, GitHub Education, etc.
6. **Referral program** — does the tool offer a personal referral link/code? What does each side get? Stackable with student discount?
7. **My referral link** — slot to fill in once I sign up, so this gets included when sharing.
8. **Official source link(s)** — direct help-center or pricing-page URLs.
9. **Last checked** — date string.
10. **Confidence** — High / Medium / Low.

## Popularity Tier Ranking (2026-05-14)

**Scope:** ranks only tools with validated student discounts (the 16 in the original audit). Referral-expansion tools (no student discount) are kept in their own section below and do NOT compete for spots here.

Ranked by general 2026 mindshare / adoption, weighted slightly toward grad-student relevance. This is the quick-scan ordering that will drive what goes on the short list in [recommended.md](./recommended.md). Tier does NOT track student-offer quality — that's a separate axis (some Tier S tools have weak student offers, some Tier C tools have very strong ones; those tradeoffs are noted per entry below).

**How to adjust:** flag any tool you want boosted/demoted based on personal familiarity or what you see classmates actually using. I'll re-tier on input.

### Tier S — Ubiquitous, mainstream AI tools
The tools everyone has heard of. If a student is using exactly one AI product, it's usually one of these.
1. **ChatGPT (OpenAI)** — global default for LLM chat. *Note: student offer is narrow — Codex credits only, no Plus discount.*
2. **Perplexity (Education Pro)** — fastest-growing AI search in academia. Strong student offer + strong referral.
3. **GitHub Copilot (Student Pack)** — gold standard for CS students; pack also unlocks ~30 other tools. *Caveat: new student sign-ups paused.*
4. **Notion (Education Plus)** — huge student user base for notes / knowledge mgmt. Free Plus is a real benefit.
5. **Adobe Creative Cloud (with Firefly)** — design / creative standard. 70% off first year.

### Tier A — Major, widely-adopted tools
Heavily used in their categories, broadly known beyond their niches.
6. **Cursor** — explosive 2024–2026 growth among devs and AI-native PM/strategy folks. Full Pro free for 1 year is a top-3 offer in this list.
7. **Figma** — design-world standard; full Pro free for higher-ed students.
8. **Claude (Anthropic)** — second-most-popular general chat AI. *No individual student discount — institutional only.*
9. **Grammarly** — large student user base for writing. Discount is seasonal/campaign-based.
10. **Microsoft Copilot (Office 365 Education)** — wide reach via school tenants. *Institution-dependent.*

### Tier B — Category leaders in popular niches
Strong adoption inside a specific use-case; less universal name recognition.
11. **Otter.ai** — leading consumer transcription tool. 20% off Pro + working referral program.
12. **Granola** — fastest-growing AI notetaker in startup / B-school circles. 12 months free Business is generous.
13. **BizCrush** — real-time in-person meeting agent with multi-language live translation + follow-up email intelligence. Newer / smaller user base than Granola, but specialty differentiation is real and 5-months-free student code makes it actionable now. *Boosted on personal-use signal.*
14. **QuillBot** — heavy student usage for paraphrasing/grammar.
15. **Canva** — massive overall user base. *Higher-ed access is institution-led Canva Campus.*

### Tier C — Specialty / growing
Real user bases but narrower scope; smaller than the names above.
16. **Zed** — AI-native dev editor; smaller user base than Cursor but gaining.
17. **Consensus** — citation-grounded academic search; strong in research-heavy programs.
18. **SciSpace** — academic research / paper Q&A. *Student policy pending verification.*
19. **Midjourney** — leading image-gen tool. *No student discount.*

### Tier D — Specialty, smaller
Niche tools, lower general awareness.
20. **Julius AI** — data analysis / chart generation. *Student policy pending verification.*
21. **Humata** — chat-with-long-PDFs. *Student policy pending verification.*

**Referral-expansion tools (Suno, Runway, Gamma, Superhuman, Fathom, Krisp, Bolt.new, HeyGen, Windsurf, etc.) are NOT ranked here** — they don't have validated student discounts, so they sit in a separate axis. See [Peer-Referral Expansion Tools](#peer-referral-expansion-tools-audit-2026-05-14) below for their own ranked picks.

Five cleanest mechanics + highest grad-student relevance:

1. **Suno** — 250 + 250 credits. AI music. Pure consumer hook.
