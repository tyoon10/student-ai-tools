# Student AI Tools — Discounts & Free Premium Access

A maintained guide to AI tools with student offers (free tiers, education discounts, referral stacking). Verified against official sources; "as of" dates recorded per entry.

## Files in this repo

- **[knowledge-base.md](./knowledge-base.md)** — internal working library. Every tool I've evaluated, including ones excluded from the public list (institutional-only, no discount, pending verification). Updated as I find new offers or terms change.
- **[recommended.md](./recommended.md)** — curated public version. Hero set of tools I actually recommend, with brief descriptions and signup links. This is the file mirrored to twyoon.com for sharing.

## Maintenance cadence

- Review quarterly or when a tool's terms visibly change (announcement, pricing page edit, expired campaign).
- Each entry carries a `Last checked` date.
- When the public list changes, update the CHANGELOG section at the bottom of [recommended.md](./recommended.md) so readers can see what moved.

## Audience

MBA students and early-career professionals looking for legitimate, individually-claimable student offers on AI tools. Not a deals-hunter list — the bar is "worth setting up an account for."
